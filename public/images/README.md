# Imagens do projeto

Use esta pasta para guardar imagens (logos, ícones, fotos, ilustrações).

**Como usar no código:**

- No HTML/JSX: `src="/images/nome-do-arquivo.png"`
- Em CSS: `url('/images/nome-do-arquivo.png')`

Arquivos em `public/` são servidos na raiz do site, então `/images/logo.png` corresponde a `public/images/logo.png`.
